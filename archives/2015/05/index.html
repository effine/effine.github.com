tail command

每次部署项目上线后，通过tomcat的日志文件catalina.out监控项目的运行情况.

$ tail -f <tomcat-dir>/logs/catalina.out       # 屏幕即不断追加打印tomcat的运行日志

当然，强制关闭打印屏幕 ctrl + c 即可.
