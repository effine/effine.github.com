屌丝逆袭修炼必备操作：
- 应用
	   command + q		# 关闭、退出程序

- finder
	   command + shift + G		# 打开“前往文件夹”，输入路径回车即可进入
	   剪切文件		# command + c -> command + option + v

- 终端
    	open + 目录		# 即可用finder打开该目录

<!-- more -->

- 截图操作
    	command + shift + 3		# 截取全屏
        command + shift + 4		# 截取自选区域
        command + shift + 4  —>释放按住的三个键 —> 空格   # 截取规则的区域
